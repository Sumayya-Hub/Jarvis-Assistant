# Jarvis Web Assistant 🌐🤖

A simple Flask-based assistant that runs in your browser. Ask questions like:
- "Tell me about Python"
- "What is the time?"
- "Open YouTube"

## 🔧 Features
- Flask web interface
- Wikipedia summaries
- Opens websites in browser
- Tells current time
